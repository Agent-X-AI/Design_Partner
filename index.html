How it works

Wrap your agents. Watch everything. Catch failures.

1

Wrap in 2 lines of code

Import the SDK. Wrap your agent call. That's it. Works with LangChain, CrewAI, AutoGen, or raw API calls. Zero config, no infrastructure changes.

2

Every output gets verified

Confidence scoring (0–1) on every response. Hallucination detection, schema validation, and task drift monitoring. Know instantly when something's off.

3

Bad outputs never reach users

Circuit breakers block low-confidence outputs. Auto-correction re-prompts the agent when it drifts. Cascade prevention stops one failure from taking down your pipeline.

agent.py
from vex import Vex # Wrap any agent in 2 lines v = Vex( confidence_threshold=0.85, auto_correct=True ) result = v.run( agent=my_agent, task="Summarize the Q4 report" ) # result.confidence → 0.94 # result.verified → True # result.corrections → 0
